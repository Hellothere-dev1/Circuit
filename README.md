# Hands-On Electronics & Microcontroller Learning Journey

This repository serves as a weekly log and documentation hub for our Applied STEM mentorship program. It tracks the progress of building hardware circuits, mastering soldering techniques, and developing firmware for microcontrollers.

---

## 🎯 Learning Objectives
* **Breadboarding:** Rapid prototyping of analog and digital circuits.
* **Soldering:** Mastering through-hole and surface-mount assembly.
* **Circuit Theory:** Understanding Ohm’s Law, components, and signal flow.
* **Embedded Systems:** Programming microcontrollers (C++/Python) to interact with the physical world.
* **Documentation:** Practicing professional engineering logs and version control.

---

## 🛠 Tech Stack & Tools
* **Hardware:** Breadboards, Multimeters, Soldering Iron, Oscilloscopes.
* **Microcontrollers:** (e.g., Arduino, ESP32, or Raspberry Pi Pico).
* **Software:** VS Code / Arduino IDE, Git, C++/C.
* **Design:** KiCad / EasyEDA (Optional for future PCB phases).
